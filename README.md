# 同途 Parallax

> 双人在线协作 2D 平台跳跃游戏 | TapTap (UrhoX 引擎)

两位玩家分别扮演"红鸟"与"黑鸟"，在同一张地图上看到截然不同的世界——红鸟眼中是温暖明亮的庆典世界，黑鸟眼中是阴冷黑暗的末日世界。玩家需要协作解谜、通过关卡，共同推进叙事。

---

## 核心特色

- **双重世界观** — 同一地图在两位玩家屏幕上呈现完全不同的视觉风格
- **在线协作** — 基于持久世界（Persistent World）的双人实时联机
- **内置地图编辑器** — 游戏内实时编辑关卡并自动同步
- **Celeste 风格物理** — 土狼时间、跳跃缓冲、长短跳区分等精细操控
- **NanoVG 矢量渲染** — 全部画面通过矢量图形管线绘制

---

## 技术栈

| 层级 | 技术选型 |
|------|---------|
| 引擎 | UrhoX (TapTap Spark Editor) |
| 脚本 | Lua 5.4 |
| 渲染 | NanoVG 矢量图形 |
| 物理 | Box2D |
| 网络 | UrhoX 内置网络层（权威服务器） |
| UI | urhox-libs/UI (Yoga Flexbox + NanoVG) |
| 云服务 | clientCloud API（好友码匹配） |

## 联机架构

- **模式**: 持久世界（Persistent World），服务器常驻，双人随进随出
- **同步**: 客户端发送控制位（移动/跳跃/交互），服务端权威判定
- **匹配**: 6 位好友码，通过 clientCloud 云变量实现跨用户房间发现

---

## 渲染系统

全部画面通过 NanoVG 矢量图形管线绘制：

1. 天空渐变 → 2. 视差背景层 → 3. 砖块纹理平台 → 4. 桥/梯/水 → 5. 物体 → 6. 角色精灵 → 7. 装饰覆盖层

**双世界配色**: `DualWorldSkin` 模块为红鸟/黑鸟提供完整独立的 RGBA 调色板，渲染器根据当前角色切换配色方案。

---

## 内置地图编辑器

在 debugSolo 模式下按 **T** 键激活，功能包括：

- 10 种编辑工具（地面、坡道、NPC、传送门、出生点、水域、桥梁、梯子、箱子、移动）
- 三种视图模式（逻辑/红鸟世界/黑鸟世界）
- 自由覆盖层系统（拖拽/旋转/缩放装饰图片）
- 完整撤销/重做
- 自动同步到服务端（0.5s 防抖热重载）
